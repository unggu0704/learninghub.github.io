### 인증

- AuthN
- 사용자가 누구인지 증명

### 권한 부여

- AuthZ
- 인증된 주체에게 작업 수행 권한 부여
- OAuth2.0 프로토콜

## 중앙 집중식 ID와 Microsoft ID

- 개별앱의 사용자 ID를 관리
