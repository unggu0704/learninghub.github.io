# Routing Algorithm

![Untitled]({{ site.baseurl }}{{ page.url }}/Routing%20Algorithm%20a8076a5e5dfa435ab92ec4ca0c4d6255/Untitled.png)

### IGP

- AS 안의 내부 타겟을 찾을떄 사용
- 다익스트라 알고리즘
- OSPF도 자주 사용함

### EGP

- AS내 다른 그룹끼리 통신
- 외부 연결을 위한 라우팅 경로 설정

### BGP

- AS 끼리 통신 할떄 그 경계에서 통신
