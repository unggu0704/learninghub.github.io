### Azure Blob Storage란?

**파일**

- 파일 서버의 역할
- 완전 관리형 파일 공유
- 온-프레미스와 동기화

**BLOB**

- REST기반 클라우드 개체 저장소
- 비정형 데이터(사진,영상)

**테이블**

- NOSQL 저장소
- 대규모 자동 크기 조절

**큐**

- Cloud Service를 통해 확장
- 메세지 가시성
